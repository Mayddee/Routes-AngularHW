About This Project

This small web application was created as a learning project to practice Angular routing, API integration, reusable components, and UI design.

The project uses the public Harry Potter API to fetch detailed information about characters from the Wizarding World. You can browse the list, search for specific characters, and view complete details for each one.

The goal of this project is simple: improve frontend skills through a clean, structured, and real API-based application.

It also serves as a practical example of how to build modern Angular pages, manage navigation, handle loading/error states, and work with external data.
